<b>"Auto Admin"</b> is a tiny accelerator framework built on top of Asp.net core.<br>Its main idea is containing reusable services that are needed across every project. <br>Also, it tries to avoid the problems that are suffered when the project gets big. (How to make the code easily maintainable)

Requirements:<br>
[.net core 5.0](https://dotnet.microsoft.com/download) <br>
[MySql 5.7](https://dev.mysql.com/downloads/mysql/5.7.html) <br>

A list of articles will be written to explain each idea and how it is implemented.

Libraries used:<br>
    As a start, this project avoids EntityFramework and uses Dapper instead.
